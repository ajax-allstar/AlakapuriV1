🏨 Alakapuri

Alakapuri is a modern full-stack web platform built to digitally transform a traditional restaurant and hotel into a smart, user-friendly, and fully online experience.

🚀 What it does

Alakapuri allows customers to explore menus, book rooms, and interact with the business online, while giving admins a centralized system to manage everything efficiently.

❌ Problem

Many restaurants and hotels lack digital systems, leading to manual work, poor customer engagement, and limited reach.

💡 Solution

Alakapuri provides a complete digital platform that simplifies bookings, improves user experience, and helps businesses move online.

✨ Features
🍽️ Interactive restaurant menu
🛏️ Room booking system
🔐 Secure authentication
📊 Admin dashboard
🤖 AI assistant (academic-focused)
🌙 Light/Dark mode
⚡ Fast & responsive design
🧑‍💻 Tech Stack

Vite + TypeScript + HTML + Tailwind CSS + Custom Backend APIs + Cloud Database + Vercel Deployment
